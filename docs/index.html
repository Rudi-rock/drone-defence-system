🚀 Getting Started

1

Clone Repository

Get the code from GitHub and navigate to the project directory.

2

Install Dependencies

Run start.bat (Windows) or start.sh (Mac/Linux) for automatic setup.

3

Start System

Backend launches on http://localhost:5000 and frontend on http://localhost:3000.

4

Run Simulation

Configure parameters and click Start to begin swarm simulation in real-time.

Quick Start Commands

Windows:

cd drone-swarm-security start.bat

Mac / Linux:

cd drone-swarm-security chmod +x start.sh ./start.sh

Docker:

docker-compose up

Then open: http://localhost:3000

🕊️ Bio-Inspiration: Boids Algorithm

The system uses Craig Reynolds' three rules from starling murmurations and fish schooling:

🚀

Separation

Avoid crowding neighbors — repulsive forces keep drones at safe distances.

🎯

Alignment

Steer toward average heading of neighbors — syncs movement across the swarm.

🧭

Cohesion

Steer toward center of mass — keeps the swarm together without gaps.
